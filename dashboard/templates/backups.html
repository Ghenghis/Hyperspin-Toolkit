Backup Tips

  • Full backups copy everything — use for first-time or major milestones.
  • Incremental backups only copy changed files — faster and smaller.
  • Every backup includes a hash manifest for verification.
  • Backups are stored at the configured backup root directory.
  • Use the Update Manager to auto-snapshot before any program update.
  • Recovery restores from any backup version to any target.
