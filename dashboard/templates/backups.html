Recovery Tips

  • Full backups copy everything — use for first-time or major milestones.
  • Incremental backups only copy changed files — faster and smaller.
  • Snapshots capture directory state with hashes for pre/post-update verification.
  • Every backup and snapshot includes a hash manifest for verification.
  • Use the Update Manager to auto-snapshot before any program update.
  • Rollbacks automatically restore the previous state if an update fails health checks.
